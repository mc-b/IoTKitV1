Mittels **RPCFunction** lassen sich Funktionen aufrufen. Dazu wird der Aufruf **run** verwendet.

#### Client 

Aufruf im Browser der entsprechenden Objekt-Methode mittels:

	http://<IP-Adresse mbed>/rpc/servos/run 0.6 0.4
